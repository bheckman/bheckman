Out of the Box Message Center

Description

Drive feature adoption by lowering the bar for entry.

Urban Airship's Message Center functionality provides a powerful platform for delivering rich content to an app's entire user base. And it's incredibly effective for brands who leverage the feature-set.

The problem? Weeks or months of development effort was often required for customer's to leverage the capability in their app.

Our goal was to design a set of broadly applicable and customizeable message center defaults as a springboard for feature adoption.

Employer: Urban Airship
Role: Lead Designer
Skills: Wireframing, Visual Design
Team: Ryan Lepinski, Marc Sciglimpaglia, Greg Weinger, Julia Duncan

Sketches

We originally set out to build several inbox styles. So I went to paper and pen to generate ideas quickly.

Those sketches were distilled into a set of default options for inbox layouts, each based on a particular image aspect ratio.
