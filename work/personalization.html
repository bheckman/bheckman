Message Personalization

Description

Timely, pertinent, and personal messaging.

Message personalization templates let you send a templated message to many users, personalized with the values that are specific to each user, e.g., first name, flight number, order status.

Building on top of the framework we created with the Message Composer, the message personalization UI allows non-technical users to easily create and update the content of a personalized message without developer intervention.
