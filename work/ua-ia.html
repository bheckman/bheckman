Information Architecture & User Flow Overhaul

Description

Making the transition to a multi-horizon company.

Urban Airship was beginning to see traction with it's digital wallet offering. Meanwhile, it was developing two entirely new add-on products.

The product had been originally built with an app-centric, single use mindset. In order to support these new offerings, the underlying information architecture and customer journey would need to be reimagined.

Employer: Urban Airship
Role: Lead Designer
Skills: User Experience Design, Visual Design, Implemenation, Information Architecture
Team: Chris Auel, Andrea Burton, Greg Weinger, Christian Metts
