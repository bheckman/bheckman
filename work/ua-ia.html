Information Architecture & User Flow Overhaul

Description

Transforming into a multi-horizon company.

Urban Airship had one product when I joined the company. It now has three product lines and four products between them.

The product had been built with an app-centric, single product mindset. Breaking this mold required a reimagining of the information architecture and user flows.

Employer: Urban Airship
Role: Lead Designer
Skills: User Experience Design, Visual Design, Implemenation, Information Architecture
Team: Soren Muus, Chris Auel, Andrea Burton, Greg Weinger, Christian Metts

Mapping User Flows

I began the project by developing a new sitemap and mapping the corresponding user interactions that allow a user to move between products and apps to accomplish their tasks with ease.
